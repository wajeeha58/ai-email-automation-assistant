# AI Email Automation Assistant 🚀

Built an AI-powered email automation workflow using Google Forms, Google Sheets, Google Apps Script, and Gmail automation.

## 📌 Project Workflow

Google Form → Google Sheets → Apps Script Trigger → Automated Email Response

✅ Form data collection
✅ Automatic response storage in Google Sheets
✅ Apps Script trigger automation
✅ Automated Gmail response workflow
✅ AI-style support reply generation

## ⚡ How It Works

1. User submits Google Form
2. Data gets stored in Google Sheets
3. Apps Script trigger executes automatically
4. AI-style email response is generated
5. Gmail sends automated reply to user
